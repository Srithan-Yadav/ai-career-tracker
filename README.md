# AI Career Tracker 🚀

A simple web app to track skills, categories, and progress levels.

## Features
- Add, edit, delete skills
- Categorize (Frontend, Backend, AI)
- Skill levels (Beginner → Advanced)
- Progress bars visualization
- Live stats dashboard

## Tech Used
- HTML
- CSS
- JavaScript (localStorage)
